##  运行环境：

htdocs本地服务器

##  账号密码：

1. 登录之前需要注册，注册成功自动跳转登录界面
2. 登录成功跳转首页网站首页

##  功能：

1. 首页：轮播图，吸顶，二级菜单，侧边导航

##  商品列表：

1. 点击商品加入购物车，
2. 点击可修改商品的数量，可删除商品

## 页面文件夹及作用:

1. css-----css样式表
2. images---图片
3. js-------js文件
4. json-----json数据
5. components----HTML文件
